HTML Interview Questions

1). What is HTML?

HTML stands for HyperText Markup Language, used to create the structure of web pages.

2). Why do we use HTML?

We use HTML to create static web pages and provide structure to a webpage, such as text, images, and links.

3). What is Doctype in HTML?

Doctype tells the browser which version of HTML the document is written in. In HTML5, it is written as `<!DOCTYPE html>`.

4). What is the root element of HTML?

The root element of an HTML document is the `<html>` tag, which wraps all the content in the document.

5). What is a tag?

A tag is used to define elements in an HTML document.uses for specific purpose. Tags can be paired or self-closing.

Types of tags:

  1. Paired Tags (e.g., <p></p>)
  2. Self-Closing Tags (or) Void Tags (e.g., <img />)
6). Give me a few examples of self-closing tags?

Examples of self-closing tags include:

  1. <input />
  2. <img />
  3. <br /> (break)
  4. <hr /> (horizontal rule)
7). What is an element?

An element is combination of starting tag, content, and ending tag.

8). How many types of elements are there in HTML?

There are two main types of elements in HTML:

  1. Block elements
  2. Inline elements
9). What is the difference between inline vs block elements?

Inline elements do not start on a new line and only take up as much width as necessary places element side by side,occupies in same line. Block elements start on a new line and occupies full width of the page.

10). What is the difference between semantic and non-semantic elements?

Semantic elements easily understable to developers and browers. (e.g., <article>, <header>, <main>, <footer>). Non-semantic elements like <div> and <span> do not describe their content.

11). What are href vs src attributes?

href is helps to navigate from pages,image,website,videos. (e.g., links), while `src` is used to gives path of the content like images,pages,website,videos etc;

12). What are the features of HTML5?

HTML5 includes new semantic elements, multimedia tags (audio, video), metatag , geolocation , canavas , form input types, and offline storage, among other features.

13). What are entities in HTML?

Entities are special characters in HTML .Those symbols are not have in laptops or computers. such as ©,®.

14). What is the difference between <div> and <span>?

<div> is a block-line element,container element, whereas <span> is an inline element, non-container element, both used to group other elements.

15). What are list tags in HTML?

There are three types of lists in html there are:1). ordered lists (<ol>),2). unordered lists (<ul>), and 3).Defintion list (<dl>).

16). What are attributes?

Attributes provide additional information about HTML elements, like `id`, `class`, `style`, `width`, `height`etc.

17). Mention a few attributes for img tags?

The attributes used in <img> tag include `src` (source), `alt` (alternative text), `width`, and `height`.

18). What is <thead> in a table?

<thead> is used to group the header content in a table or Head of the table.

19). What is <tr>?

<tr> stands for table row, used to define rows in a table.

20). What is <th>?

<th> defines a header cell in a table, typically bold and centered.

21). What is <td>?

<td> defines a data cell in a table.

22). Can I use <th> in <tbody>?

Yes, <th> can be used in <tbody>, but it is most commonly used in <thead>.

23). What are meta tags?

Meta tags provide metadata about an HTML document, such as character set, author,enhance SEO purpose,SEO keywords,To rank top of search engine.

24). What is SEO?

SEO stands for Search Engine Optimization, which improves the visibility and ranking of a website in search engine results.

25). What is an attribute of the <html> tag?

The <html> tag attribute is `lang`; tag to specify the language of the document.

26). What is an iframe and anchor?

An iframe is used to embed external source like videos,audio,images,locations within the current page, while an anchor (<a>) is used to create hyperlinks to navigate other resources.

27). What is the vs Git and GitHub?

Git is a version control system.It is an application software.Used for local storage, while GitHub is a web application ,colud based ,store git repositories online,used for project management collaboration.

28). What is the workflow of GitHub?

The GitHub workflow typically involves Working directory (git init git add) ,Staging area(git commit),Local repository(git push),Colud repository.

29). What command do we need use to move changes to staged changes?

To move changes to the staging area, you use the `git add` command.

30). How can you stage all files at a time?

You can stage all files at once by using `git add .`.

31). What is the git push command and use of it?

The `git push` command is used to send the committed changes from your local repository to a (remote) Colud repository, such as GitHub.

32). What is the git commit command usage?

The `git commit` command is used to save changes to the local repository. It requires a message to describe the changes.

33). What is the use of the git init command?

The `git init` command is used to initialize a new Git repository in a directory.

34). What are formatting tags?

Formatting tags are used to style text, such as <b> (bold), <i> (italic), and <u> (underline).

35). What are input tag attributes?

Input tag attributes include `type`, `name`, `value`, `placeholder`, `id`,`pattern`and `required`.

36). What are form attributes?

Form attributes include `action`, `method`, `target`, and `align`.

37). What is a marquee tag?

The <marquee> tag is used to create scrolling text on a webpage.

38). What are marquee tag attributes?

Attributes of the <marquee> tag include `direction`, `behavior`, `scrollamount`, and `loop`.

39). What are multimedia tags?

Multimedia tags in HTML include <audio> and <video> to embed audio and video content.

40). Can we use block elements in inline elements?

No, block elements should not be placed inside inline elements.

41). What is the difference between absolute and relative paths?

An absolute path uses external sources,while relative path uses inside the source of computer.

42). What are summary and details tags?

<summary> defines a visible heading, and <details> is used to create expandable content sections.

43). What tag is helpful to give the reviews?

To give reviews we use Textarea tag.

44). What are the fieldset and legend tags?

The <fieldset> tag used elements in a form, and the <legend> tag provides a caption for the group.
